hours

round

Returns the result of rounding time/Duration to the nearest multiple of duration. The rounding behavior for halfway values is to round away from zero. If the result exceeds the maximum (or minimum) value that can be stored in a Duration, round returns the maximum (or minimum) duration. If duration <= 0, round returns time/Duration unchanged.

(time/Duration round duration) -> duration

truncate

Returns the result of rounding time/Duration toward zero to a multiple of duration. If duration <= 0, truncate returns time/Duration unchanged.

(time/Duration truncate duration) -> duration
